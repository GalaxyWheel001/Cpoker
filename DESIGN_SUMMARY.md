## 🔧 Использование

### Быстрый старт
```jsx
// Градиентная кнопка
<Button variant="gradient" iconName="play">
  Присоединиться
</Button>

// Стеклянная карточка
<Card variant="glass" hover>
  <CardHeader>
    <CardTitle>Название</CardTitle>
  </CardHeader>
  <CardContent>
    Содержимое
  </CardContent>
</Card>

// Статусный бейдж
<StatusBadge status="online" />
```
